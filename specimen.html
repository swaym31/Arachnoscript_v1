Set in Arachnoscript Text. The letters and numbers are the custom glyphs; the comma, parentheses, ampersand, quotes, dash and percent come from the English base font built into the same file.

USE

Drop it into your keypad

Put a font file next to your CSS and load it. Arachnoscript (your glyphs only) keeps the file tiny — leave punctuation out and let the browser fall back to a system font:

@font-face {
  font-family: 'Arachnoscript';
  src: url('assets/Arachnoscript-Regular.woff2') format('woff2'),
       url('assets/Arachnoscript-Regular.ttf')   format('truetype');
  font-display: swap;
}

/* letters & digits → your glyphs; commas, etc. → fallback */
.arachnoscript { font-family: 'Arachnoscript', system-ui, sans-serif; }

Prefer one self-contained file that needs no fallback (Word, exports, anywhere)? Use ArachnoscriptText-Regular.woff2 instead — punctuation is built in, so font-family:'Arachnoscript Text' alone is enough.
